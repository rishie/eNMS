The skip query parameter allows you to define python expression to determine if the service should be skipped.

This expression is evaluated to determine what to SKIP - if you happen to use an expression that identifies what you WANT - then you will likely have to negate that expression.

If a service is skipped, the Skip Value determines the workflow result (success = True, False). The default value is True.

Variable substitution can be performed in this block.
